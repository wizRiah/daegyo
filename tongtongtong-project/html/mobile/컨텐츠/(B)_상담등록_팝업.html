상담(등록) 알림발송
- 상담 등록/수정 시, 상담(등록) 알림발송 이력이 없는 경우에 한하여 선택된 발송유형으로 하단 알림발송대상에게 상담결과가 포함되지 않은 상담(등록) 알림발송 내용이 최초 1회 발송됩니다. - 추후 별도의 상담결과를 포함한 상담(결과) 알림발송 1회 가능


상담(결과) 알림발송
- 상담 등록/수정 시, 상담(결과) 알림발송 이력이 없는 경우에 한하여 상담결과에서 ‘결과입력‘ 체크 후 상담결과 내용이 작성된 경우 선택된 발송유형으로 하단 알림발송대상에게 상담결과가 포함된 상담(결과) 알림발송 내용이 최초 1회 발송됩니다.


자동알림발송 유의사항
- ‘발송유형’, ‘알림발송대상’ 이 선택되지 않은 경우 발송되지 않음


알림발송내용 설정 및 확인
- 알림발송 내용은 환경설정/알림설정에서 확인 및 설정이 가능합니다.
