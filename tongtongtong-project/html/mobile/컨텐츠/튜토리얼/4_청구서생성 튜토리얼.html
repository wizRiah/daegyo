청구서 생성Check Point
  • 분류선택
    청구서 생성 월은 등록한 반(클래스) 정보 중 수납방식이 월 방식인 경우에 해당됩니다.
    청구서 생성 월은 당월로 입력해 주세요.
    청구서 생성 월은 1번만 입력하면 되고 년도와 월만 입력하면 됩니다. (예시 YYYY-MM)
  • 분류 / 반 선택에서는 청구서를 일괄 생성할 반을 선택합니다.
    중분류 하위에 반을 선택하고 화면 하단의 “수강료청구서일괄생성”버튼을 클릭하면 해당 반에 청구서가 일괄 생성됩니다.
    반 전체 적용 체크박스를 선택 후 “수강료 청구서일괄생성”버튼을 클릭하면 등록한 모든 반에 청구서가 일괄 생성됩니다.
    • 수강료 : 등록중인 반의 수강료를 직접 입력합니다.
      (숫자만 입력가능)
    · 수강하는 반의 수강료
    반(클래스) 등록 시 반 정보에 입력한 수강금액 으로 청구서가 일괄 생성되도록 선택하는 조건 입니다.
    · 직접입력(일괄적용)
    이미 반 정보에 수강료가 입력되어 있으나, 직접입력 선택 후 수강금액을 입력하면, 해당 금액으로 청구서가 일괄 생성됩니다.
    직접입력 (일괄적용) 선택 시 당월에는 반에 등록된 수강금액이 무시되고 직접 입력된 금액이 우선 시 됩니다.
