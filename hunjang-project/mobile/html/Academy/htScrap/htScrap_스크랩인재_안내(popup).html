공개제의한 인재 안내

  • 인재정보를 통해 수집된 개인정보 및 이력서는 개인정보보호정책에 의거 인재채용의 목적으로만 이용하셔야 합니다.
  • 제안을 보낸 이력서는 발송일로부터 1년간 보관되며, 연락처와 상세정보는 90일간 확인 할 수 있습니다.
  • 연락처 비공개 인재가 연락처 공개 제안을 수락하면 연락처가 포함된 이력서를 확인 할 수 있습니다.
  • 연락처 비공개 인재가 연락처 공개 제안을 수락하기 전, 여러 번의 제안을 보낼 때에 서칭 건수가 추가로 차감되지 않습니다.
  • 포지션 제안 시 열람건수가 차감되지 않고, 연락처 공개 제안 수락 시 건수가 차감됩니다.
  • 거절이나 미응답 회원이 이력서를 비공개로 변경하거나 삭제한 경우 이력서를 확인 할 수 없고, 공개 이력서를 변경한 경우 변경된 이력서가 보여집니다.
